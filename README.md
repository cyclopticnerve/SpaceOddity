## Downloading

There are two ways to get the code:

1. Download the [latest
release](https://github.com/cyclopticnerve/SpaceOddity/releases/latest)
(the **'Source code (zip)'** file should work an all platforms).

2. Or you can clone the git repo to get the latest (and often broken) code from
   the main branch:
```bash
$ git clone https://github.com/cyclopticnerve/SpaceOddity
```

## Installing

<!-- __RM_APP__ -->
If you downloaded the zip file, extract it and go to the 'install' directory:
```bash
$ cd ~/Downloads/SpaceOddity-<version>/install
```
where \<version\> is the version number of the file you downloaded.

If you cloned the repo, just go to the 'install' directory:
```bash
$ cd ~/Downloads/SpaceOddity/install
```

Either way, run the install script:
```bash
$ ./install.py
```

## Uninstalling

<!-- __RM_APP__ -->
Go to the source folder for SpaceOddity and run the uninstaller:
```bash
$ cd ~/.local/share/spaceoddity
$ ./uninstall.py
```
